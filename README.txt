This plugin is providing usefull libs to develop/deploy JSON APIs.

It should be usefull to have a look into :
- lib/action/
- lib/accessor/
- lib/services/

This abstract plugin is already used by liOnlinceChoicesPlugin, if you need some code examples.

INSTALL:
1. add data in your config/extra-plugins.php file using the data in the %SF_PLUGIN_DIR%/config/extra-plugins.php.template
